Web Tools for UC Santa Cruz

Use these tools to edit images, build callout boxes, and create buttons and banners for UC Santa Cruz websites, articles, and email messages.

  • Image Resizing and Cropping

    A tool to crop images into sizes used on UCSC websites, articles, and emails.

  • Button Builder

    A tool to generate email and form buttons.

  • Giving Button Builder

    A tool to generate giving buttons for web pages.

  • Callout Box Builder

    A tool to generate callout boxes for WCMS web pages.

  • Email Banner Builder

    A tool to generate email banners for UCSC emails.

  • Changelog

    A list of posts detailing updates to this site
